## Cara Menjalankan Notebook

### Menggunakan Google Colab

1. Buka notebook yang diinginkan.
2. Klik tombol **Open in Colab** (jika tersedia).
3. Jalankan notebook dari sel pertama hingga terakhir.

### Menjalankan Secara Lokal

1. Clone repository:

```bash
git clone https://github.com/sherlyekawindiani/data-science-2026.git
```

2. Masuk ke folder repository:

```bash
cd data-science-2026
```

3. Install library yang diperlukan:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

4. Jalankan Jupyter Notebook:

```bash
jupyter notebook
```
